## Kia Soul (Gen 2, PS, 2014-2016) — 1.6 GDI 132 PS

| Thema | Details |
|-------|---------|
| **Hinterachs-Rost** | Verbundlenkerachse rostet von innen (Wasser+Salz im Hohlprofil). Unbemerkt bis Achse bricht. |
| Erkennung | Hammer-Test: Metallklang = gut, dumpf = Rost. Rostfahnen an Schweißnähten prüfen. |
| Kosten Fix | Neue Achse Original ~1.200-1.500€ + Einbau 500-800€ = ~1.700-2.300€. Gebrauchtachse ~300-400€ + Einbau. |
| Kulanz | In UK/NL Rückruf. In DE: Kulanz auf Kulanz — nicht planbar. |
| Steuerkette | 1.6 GDI Gamma: hält gut, kein Serienproblem. |
| DSG | Kein DSG — entweder 6-Gang Handschalter oder 6-Gang Automatik (Wandler, robust). |
| Euro | 6 → Steuer ~90€/J |
| NCAP | 5 Sterne (2014) |

## Renault Kangoo Trekka 4x4 (2002-2007)

- Echter Allrad (Nissan ATC-Kupplung + ASR), kein Höherlegungskit
- Bodenfreiheit ~190mm, Wattiefe 400mm
- 1.6 16V 110 PS (9L/100km) oder 1.9 dCi 80 PS (7L/100km)
- Vollständig unabhängige Hinterachse (keine Verbundlenker)
- **19-23 Jahre alt** — Rost überall (Schweller, Radläufe, Schiebetüren)
- Ersatzteile Allrad-Komponenten werden knapp
- 3 NCAP, kein ESP, 2 Airbags
- Euro 3/4 → Umweltzonen
